You are given an array a of n integers and an integer k.

In one operation, you can choose an index i (1 ≤ i ≤ n) and increase a[i] by 1.

Your goal is to make at least one subarray of length at least 2 such that the greatest common divisor of all elements in that subarray is divisible by k.

Output the minimum number of operations required.

Input:
The first line contains two integers n and k (2 ≤ n ≤ 2×10^5, 1 ≤ k ≤ 10^9)
The second line contains n integers a[i] (1 ≤ a[i] ≤ 10^9)

Output:
Print a single integer — the minimum number of operations.
